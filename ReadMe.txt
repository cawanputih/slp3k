Import database ke phpmyadmin dari folder database. Terdapat dua file sql yaitu 
tabelprokersql dan tabelprogressql.Database name untuk kedua tabel adalah spl3kdb.
Setelah xampp start, prototipe dijalankan dengan menelusuri "localhost/spl3k" di address bar browser
User akan melihat tampilan daftar program kerja yang telah ada.
Dari tampilan tersebut dapat dipilih , apakah ingin menambahkan program kerja, 
melakukan generate laporan atau memilih program kerja yang telah ada.
Ketika suatu program kerja di pilih, maka akan menuju tampilan
daftar progres program kerja tersebut.

link kolaburasi "https://github.com/cawanputih/slp3k"
