Import database ke phpmyadmin dari folder database. Terdapat dua file sql yaitu 
tabelprokersql dan tabelprogressql.Database name untuk kedua tabel adalah spl3kdb.
Setelah xampp start, prototipe dijalankan dengan menelusuri "localhost/spl3k" di address bar browser
User akan melihat halaman login. 
*Untuk ketua yayasan login dengan username = "ketuayayasan" password ="ketuayayasan"
*Untuk ketua divisi login dengan contoh username = "ketuadivisihumas" password ="ketuadivisihumas"

User akan melihat tampilan sesuai dengan login dengan username yang bersangkutan.
Ketua yayasan, akan dapat melihat program kerja setiap divisi, sedangkan ketua divisi , hanya 
dapat melihat program kerja milik divisinya sendiri.

Untuk ketua yayasan dapat memilih divisi, program kerja di divisi tersebut,
 beserta progres pada program kerja yang dipilih. Ketua yayasan juga dapat melakukan generate laporan
 per divisi.

Untuk ketua divisi dapat memilih , apakah ingin menambahkan program kerja, 
melakukan generate laporan atau memilih program kerja yang telah ada.
Ketika suatu program kerja di pilih, maka akan menuju tampilan
daftar progres program kerja tersebut yang dapat ditambah, edit, maupun hapus.
